## Variables
This is where we refer something using something else.

> when we say x-= 23 23 is x and x is 23
> we use let or const
> Never use var at all

    let x=23;
    let y=50;
    const z=50;

Rules for variables

 1. Space is not allowed
 2. Only letters, digits, underscores and dollar signs are permitted.
 3. Case matters in variable names
 4. letter, underscore, $ can only begin in a variable.

Pascal Case.
first letter of each word is capital from the beginning 
used in JS
//MyAgeInTheNext5Years=45

Camel case.
same with pascal but first word starts with small letter
//myAgeInTheNext5Years=45

Snake case.
use of (_)
//my_age_in_the_next_5_years=45
